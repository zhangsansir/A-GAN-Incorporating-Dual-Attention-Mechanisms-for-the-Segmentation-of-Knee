main.py: Parameter configuration and program entry point.
ops.py and utils.py: Modules containing utility functions and common operations.
environment.yml: Environment configuration file for dependency setup.
se_skip_gate.py: Model implementation that integrates SE attention, Attention Gate (AG), and multi-scale mechanisms.
zys_res2rgb_oai.m: Converts segmentation results generated by the model into single RGB color images.
zys_rgb2tissue_oai.m: Extracts individual tissue regions from RGB images.
zys_mat6_oai.m: Dataset preparation script.
gan_chushi.py: Original GAN model implementation.
gan_msg.py: GAN model with a multi-scale mechanism.
gan_ag.py: GAN model with Attention Gate (AG).
gan_se.py: GAN model with SE attention mechanism.
gan_se_ag.py: GAN model combining SE attention and Attention Gate mechanisms.
